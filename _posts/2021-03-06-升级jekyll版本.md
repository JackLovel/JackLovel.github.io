> 我的环境
先说一下我的环境吧：
```
ruby: 3.0.0
```

> 更改[Gemfile](https://github.com/JackLovel/JackLovel.github.io/blob/master/Gemfile) 

- 其中需要添加 
```
kramdown-parser-gfm
github-pages
webrick
```

- 还要升级下面两个gem 版本
```
gem "jekyll", "~>3.9.0" 
gem "kramdown", "~>2.3.0"
```

> 更新并运行
```
> bundle install
> bundle exec jekyll serve
```
